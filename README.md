1. Install dependencies:

```bash
npm install
```


2. Create a `.env` file in the root directory with your API:

```
BINANCE_API_KEY=
BINANCE_SECRET_KEY=

BYBIT_API_KEY=
BYBIT_SECRET_KEY=

KUCOIN_API_KEY=
KUCOIN_SECRET_KEY=
KUCOIN_PASSPHRASE=

HTX_API_KEY=
HTX_SECRET_KEY=

OKX_API_KEY=
OKX_SECRET_KEY=
OKX_PASSPHRASE=

HYPERLIQUID_API_KEY=
HYPERLIQUID_SECRET_KEY=
HYPERLIQUID_WALLET_ADDRESS=
```

3. Start:

```bash
npm start
